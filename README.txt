# Twitter Sentiment Analysis

## Overview

This repository contains two versions of a Twitter Sentiment Analysis project. The project utilizes the Tweepy library to fetch tweets and analyzes their sentiment using TextBlob. The analysis results are visualized using Matplotlib.

## Versions

1. **Main Version**: Contains the complete implementation.
2. **Free Version**: A slightly modified version with adjusted features.

## Features

- Fetches live tweets using Tweepy API
- Performs sentiment analysis (positive, negative, neutral)
- Visualizes sentiment distribution using Matplotlib

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/twitter-sentiment-analysis.git
   cd twitter-sentiment-analysis
   ```
2. Install dependencies:
   ```bash
   pip install tweepy textblob pandas matplotlib
   ```

## Usage

1. Open the Jupyter Notebook:
   ```bash
   jupyter notebook
   ```
2. Load `main.ipynb` or `main_free_version.ipynb`
3. Replace the placeholder API keys in the script with your own.
4. Run the cells to analyze Twitter sentiment.

## Note

- Ensure you have valid Twitter API credentials before running the notebook.
- Keep your API keys secure and do not share them publicly.
